📊 PeopleIQ – AI-Powered Employee Intelligence Platform

🚀 Overview
PeopleIQ is a smart, AI-powered employee intelligence system designed to bridge the gap between HR decision-making and employee career growth.

It provides a dual-dashboard system:

👨‍💻 Employee Portal (Self-growth & insights)
👔 HR Portal (Organizational intelligence & analytics)

The platform goes beyond tracking — it analyzes, predicts, and guides employee potential.

✨ Key Features

👨‍💻 Employee Dashboard

📌 1. Personal Dashboard (/employee/dashboard)

* Profile completion progress tracker
* Top skills overview
* Recent achievements
* Smart action suggestions to improve profile score

🧩 2. Profile Builder (/employee/profile)
Step-by-step profile creation system:

* Personal Information

* Education

* Skills

* Experience

* Certifications

* Projects

* Behavioral Patterns

* Career Goals

* Incremental saving to SQLite database

* Structured data for AI analysis

📈 3. Career Hub (/employee/career)

* Tracks career trajectory
* Suggests internal promotion opportunities
* Identifies skill gaps for target roles

📊 4. My Insights (/employee/insights)
Personalized analytics dashboard:

* Strength analysis
* Role-fit scoring
* Peer comparison metrics
* Visual representation of growth

🛠️ 5. Support Center (/employee/support)

* Raise support tickets
* Access FAQs
* Direct communication with HR

🤖 6. AI Coach (Global Assistant)

* Persistent AI chat assistant
* Helps with:

  * Profile completion
  * Career guidance
  * Skill improvement suggestions
* Real-time interactive support

👔 HR Portal Features

📊 1. Master Dashboard (/hr/dashboard)
Organization-wide KPIs:

Total employees
Average profile completion
Burnout risk indicators
High-potential talent identification
Department-wise analytics

📂 2. Employee Directory (/hr/directory)

Centralized employee database
Advanced search & filters:
Department
Location
Risk level
Quick access to employee profiles

🔍 3. Deep-Dive Employee Profile (/hr/employee/)

Complete employee lifecycle view
Displays:
Skill badges

AI insights:
